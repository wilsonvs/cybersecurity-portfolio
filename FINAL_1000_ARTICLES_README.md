1. Navigate to **Zoho Desk → Knowledge Base → Articles**
2. Click **Import** or **Bulk Import**
3. Upload `CYBERPHORE_1000_FULL_CONTENT.csv`

### Step 3: Map Fields
```
CSV Column          →    Zoho Desk Field
─────────────────────────────────────────
Article Title       →    Title
Slug                →    Permalink/SEO URL
Category            →    Category
Sub-Category        →    Sub-Category
Owner               →    Author/Owner
Tags                →    Tags/Keywords
Body (HTML)         →    Article Body/Content
Meta Description    →    SEO Description
```

### Step 4: Import Settings
- **Encoding:** UTF-8
- **Status:** Published (or Draft for review)
- **Visibility:** Public or Internal
