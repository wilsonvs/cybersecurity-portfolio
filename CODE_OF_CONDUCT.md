# Code of Conduct

## Purpose

This portfolio is intended to support professional cybersecurity learning and collaboration. Discussions and contributions should be respectful, constructive, and focused on improving security knowledge.

## Expected Behavior

- Use professional and respectful language.
- Provide constructive feedback.
- Protect privacy and avoid sharing sensitive data.
- Keep examples legal, ethical, and lab-safe.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Sharing credentials, secrets, private logs, or personal data.
- Encouraging unauthorized access or harmful activity.
- Submitting copied work without proper attribution.

## Enforcement

Issues or contributions that violate this code may be closed, edited, or removed.
