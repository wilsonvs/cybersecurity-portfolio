# Contributing

This repository is maintained as a personal cybersecurity portfolio. Contributions are welcome when they improve technical accuracy, clarity, structure, or professional presentation.

## Contribution Guidelines

- Keep writing clear, concise, and professional.
- Use sanitized lab data only.
- Avoid copyrighted course content unless it is properly referenced and allowed.
- Prefer practical security examples over generic notes.
- Add references for frameworks, tools, or standards when useful.

## Suggested Improvements

- Fix typos or unclear wording.
- Improve Markdown formatting.
- Add diagrams or tables that make security workflows easier to understand.
- Suggest better project structure for SOC, SIEM, GRC, or vulnerability management content.
