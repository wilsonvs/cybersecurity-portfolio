## Import Instructions for Zoho Desk

### Step 1: Prepare Import
1. Download `cyberphore_knowledgebase_FINAL.csv`
2. Ensure Zoho Desk categories match the CSV categories (or create them)

### Step 2: Import to Zoho Desk
1. Navigate to **Zoho Desk > Knowledge Base > Articles**
2. Click **Import Articles**
3. Select the CSV file: `cyberphore_knowledgebase_FINAL.csv`
4. Map CSV columns to Zoho Desk fields:
   - Article Title → Title
   - Slug → Permalink/SEO URL
   - Category → Category
   - Sub-Category → Sub-Category
   - Owner → Author/Owner
   - Tags → Tags
   - Body (HTML) → Article Body/Content
   - Meta Description → SEO Description

### Step 3: Verify Import
1. Check that all 163 articles imported successfully
2. Verify categories and tags are properly assigned
3. Ensure images are displaying correctly
4. Test article search and navigation

### Step 4: Post-Import Tasks
1. Review and publish articles (set to published status)
2. Configure article permissions if needed
3. Set up related articles/cross-linking
4. Enable user feedback/ratings
5. Configure SEO settings in Zoho Desk
